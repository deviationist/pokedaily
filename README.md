# PokéDaily

A mobile-friendly web app for discovering, searching, and collecting Pokémon. Discover a random Pokémon each visit, search the full Pokédex by name, and build your personal collection — all persisted locally in your browser.

Built with React, TypeScript, and Vite. Powered by [PokéAPI](https://pokeapi.co/).

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:5173`.

## Scripts

| Command             | Description                          |
| ------------------- | ------------------------------------ |
| `npm run dev`       | Start dev server with hot reload     |
| `npm run build`     | Type-check and build for production  |
| `npm run preview`   | Preview the production build locally |
| `npm run lint`      | Run ESLint                           |
